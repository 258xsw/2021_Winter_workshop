2021_JARAM_Winter_Workshop
==========================
마크다운 사용법
---------------
# 1. 마크다운이란
> markdown      
* 텍스트 기반의 마크업언어    
* 2004년 존그루버가 만들었음.    
* 쉽게 쓰고 읽을 수 있다.    
# 2. 장점    
- 간결하다.
- 별도의 도구없이 작성가능하다.
- 다양한 형태로 변환이 가능하다.
- 텍스트(Text)로 저장되기 때문에 용량이 적어 보관이 용이하다.
- 텍스트파일이기 때문에 버전관리시스템을 이용하여 변경이력을 관리할 수 있다.
- 지원하는 프로그램과 플랫폼이 다양하다.  
# 3. 단점    
+ 표준이 없다.
+ 표준이 없기 때문에 도구에 따라서 변환방식이나 생성물이 다르다.
+ 모든 HTML 마크업을 대신하지 못한다.
